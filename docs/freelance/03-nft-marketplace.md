# NTF Marketplace

> 2021

An MVP, which is a marketplace for buying/selling/renting game assets using Etherium blockchain.

- Teamsize: 8
- Period: 1 month

## What I did

- Handle connecting app with Metamask wallet
- Implement UI base on designs
- Write database queries
- Integrate API

## What I used

- Nextjs
- Prisma
- PostgreSQL
- Web3js
